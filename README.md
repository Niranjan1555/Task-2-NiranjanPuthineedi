# Data Classification Using AI

## Description

This project is developed as part of the DecodeLabs Industrial Training Program (Week 2). The objective is to build a machine learning classification model using the Iris dataset and the K-Nearest Neighbors (KNN) algorithm.

## Features

- Load the Iris dataset
- Split data into training and testing sets
- Train a KNN classifier
- Predict classes for unseen data
- Calculate accuracy
- Generate a confusion matrix
- Display classification report
- Predict flower species for new samples

## Technologies Used

- Python 3
- Scikit-learn
- KNeighborsClassifier
- train_test_split

## Dataset

Iris Dataset

## Evaluation Metrics

- Accuracy Score
- Precision
- Recall
- F1 Score
- Confusion Matrix
